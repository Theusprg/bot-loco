# 🤖 Bot Loco - Auto Mensageiro

Este é um bot com interface gráfica feito em Python que automatiza o envio de mensagens em um site específico, utilizando Selenium.

---

## 📦 Funcionalidades

- Interface simples com Tkinter
- Envio automático de mensagens aleatórias de um arquivo `.txt`
- Delay aleatório entre envios (tempo mínimo e máximo configuráveis)
- Anti-detecção básico (modificação do `user-agent` e `webdriver`)
- Carregamento de link personalizado para qualquer site compatível

---

## 🖼️ Interface

A interface permite:

- Informar o link do site
- Definir tempo mínimo e máximo entre mensagens
- Carregar um arquivo `.txt` com frases
- Iniciar e parar o bot facilmente
